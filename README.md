# Keen Keeper

Live Demo: https://keen-keeper-f3ks.vercel.app/

Overview:
Keen Keeper is a task management web application that allows users to create, update, and delete tasks efficiently. It provides a simple and responsive interface to organize daily work and improve productivity.

Task management apps help users organize, track, and manage tasks in one place, making it easier to stay focused and complete work efficiently. :contentReference[oaicite:0]{index=0}

Technologies:
- React.js
- JavaScript (ES6)
- Tailwind CSS

Features:
- Add new tasks
- Update existing tasks
- Delete tasks
- Real-time UI updates
- Responsive design for all devices

Dependencies:
- react
- react-dom
- tailwindcss

Run Locally:
git clone https://github.com/masumgaibandha/keen-keeper.git
cd keen-keeper
npm install
npm run dev
