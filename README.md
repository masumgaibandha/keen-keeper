# Friend Keeper App

## Description
This is a simple Next.js app where I track my interactions with friends like calls, texts, and video chats. I can see them in a timeline and view stats.

## Technologies
- Next.js
- React
- Tailwind CSS + daisyUI
- Recharts
- LocalStorage
- React Icons

## Features
- Add Call, Text, Video interactions
- View all activities in a timeline
- Filter timeline by type (Call, Text, Video)
